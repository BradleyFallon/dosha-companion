# Notes Inbox

Capture ideas here without organizing them first. Short fragments are acceptable.

During planning triage:

1. Delete irrelevant notes.
2. Merge duplicate ideas.
3. Promote meaningful work to `planning/backlog/` using `planning/templates/work-item.md`.
4. Add a type and priority.
5. Move only current work to `planning/active/`.

Do not treat this file as a specification or durable decision record.
