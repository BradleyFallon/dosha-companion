# Architecture Overview

> Demo implementation note: the current application is a static React/Vite client. Quiz CSVs and editor-owned Markdown/JSON are validated at generation time into typed TypeScript modules. A small repository layer serves content to screens. Profile, assessment, recommendation history, and dated check-ins are validated in browser-local persistence v4. There is no backend, remote account, CMS, AI service, or cloud synchronization.
