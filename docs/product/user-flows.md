# User Flows

## 1. Create an account

The new user signs up, accepts required terms, verifies their account if required, and enters onboarding.

## 2. Complete a basic profile

The user provides required onboarding information, reviews how it will be used, and may skip fields marked optional.

## 3. Answer initial questions

The user completes the initial assessment, may resume later, and sees progress and plain-language context throughout.

## 4. Unlock a preliminary dosha profile

After the minimum answer threshold, the user receives a clearly labeled preliminary baseline profile, current-balance summary, confidence indicator, and next steps.

## 5. Receive daily guidance

The returning user opens Today, sees relevant expert-approved content and practical actions, and can open related learning material.

## 6. Answer additional questions over time

The user answers prioritized refinement or check-in questions. The product explains when results change and distinguishes baseline tendencies from recent conditions.

## 7. Ask the premium assistant a question

An eligible subscriber asks a question. The assistant uses approved content and permitted user context, cites or links relevant learning material, and follows safety rules. A non-subscriber sees a subscription prompt.

## 8. Create and publish a daily post

An editor drafts and tags a post, an Ayurvedic expert reviews it, required safety metadata is added, an authorized reviewer approves it, and the editor schedules or publishes it with revision history retained.
