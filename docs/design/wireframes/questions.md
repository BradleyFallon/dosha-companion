## No suggested questions

```text
+--------------------------------------+
| You’re up to date                     |
| No questions are especially useful   |
| right now. Check back later or        |
| explore an optional topic.            |
|                                      |
| [ Explore optional questions        ]|
+--------------------------------------+
```

## Notes

- Use type labels and time windows to distinguish refinement from current check-ins.
- Do not show an endless remaining-total count.
- “Answer more” is optional and visually secondary.
- Freshness describes recent data, not whether the user has maintained a streak.
